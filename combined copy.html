Help
Bug Reports
Submit new issues on the GitHub repository. Before you submit a bug report please follow the following steps:
  1. Make sure the bug can be consistently reproduced
  2. Search open issues to make sure the bug hasn't been reported before
  3. Attach the error message from inspect element console if possible.
Feature Requests
Submit new requests on the GitHub repository. Before you request a feature, search open issues to make sure it hasn't been requested before.

Local Development
#contributing
To set up local development, follow the instructions below. This project was designed specifically to be completely independent and dependency free. Refer to the guide below to learn how to set up your own personal copy.
This project is regularly updated. Use GitHub's fork feature to merge the latest changes into your fork. If you have not changed any of the code other than the contents of the HTML file, the updates should gracefully merge into your existing fork. On the contrary, if any changes were made to the CSS or JS, you might need to carefully merge the updates yourself.
